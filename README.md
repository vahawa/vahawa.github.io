# BVG Abfahrten Express

Echtzeit-Abfahrtsmonitor für den Berliner Nahverkehr (BVG/VBB).

**Live:** [vahawa.github.io](https://vahawa.github.io/)

## Features
- GPS-basierte Haltestellenerkennung
- Echtzeit-Abfahrten via BVG API
- Haltestellenfilter, Radius- und Anzahl-Einstellungen
- Auto-Refresh alle 30s
- Dark/Light Mode (System)
- Installierbar als Home-Screen-App (iOS/Android)
- Komplett client-seitig, kein Backend
